# SOFI Technologies Valuation Model

## Overview
This project develops a discounted cash flow (DCF) valuation model for SOFI Technologies using projected financial performance through 2029. The model forecasts revenue growth, profitability, lending activity, deposits, and free cash flow to estimate an implied share price under multiple valuation scenarios.

## Model Features
- Historical financial analysis
- Revenue and EBITDA forecasting
- Loan and deposit growth projections
- Free cash flow estimation
- Discounted cash flow (DCF) valuation
- Sensitivity analysis on WACC and terminal growth assumptions
- Investment recommendation summary

## Key Assumptions
- Revenue growth gradually declines from 20% to 14%
- EBITDA margins expand from 31% to 34%
- WACC: 9.0%
- Terminal growth rate: 2.5%

## Valuation Summary
- Implied share price: $18.67
- Current share price (5/6/2026): $16.30
- Estimated upside: 14.5%
- Recommendation: Moderate Buy

## Tools Used
- Microsoft Excel
- Financial modeling
- DCF valuation
- Sensitivity and scenario analysis
